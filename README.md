Turn your boring browser into a emoticon face that responds to the sounds around you. Huzzah!

To get started:

 - `npm install`
 - `coffee app.coffee`
 - Visit http://localhost:8080/.
 - Allow the browser to use your microphone.
 - Make some noise.
